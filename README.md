# Codeforces Solutions 🚀

This repository contains my solutions to problems from [Codeforces](https://codeforces.com/). I use this space to track my progress in competitive programming and improve my problem-solving skills in Data Structures and Algorithms.

* **Language:** Java 21

## 📂 Repository Structure
Each problem has its own folder named with the **Problem ID** and **Name** (e.g., `617A - Elephant`).
Inside, you will typically find:
* `Main.java` - The accepted solution source code.
* (Optional) Alternative approaches or optimized versions.

## 🛠️ How to Run
To run any solution locally:
1.  Navigate to the problem folder.
2.  Compile the Java file: `javac Main.java`
3.  Run the class: `java Main`
4.  Input the test case data.
